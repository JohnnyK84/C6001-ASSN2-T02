Resources and Documentation

TOOLS AND RESOURCES:

A number of online tools and resources are utilized in the development and operation of this application.

Geographical data and the Google maps are obtained from the Google Maps APIs site:
https://maps.googleapis.com/maps/api

Weather data is obtained from the Open Weather Map APIs site:
http://api.openweathermap.org/data

Pre-built Electron, package JSON, and jquery library script are relied on for processing and presenting the data.
