Tools and Resources
A number of online tools and resources are utilized in the development and operation of this app. Geographical data is obtained from the Google Maps APIs site: https://maps.googleapis.com/maps/api Weather data is obtained from the Open Weather Map site: http://api.openweathermap.org/data Pre-built Electron, package JSON, and library jquery script are relied on processing and presenting the data.
